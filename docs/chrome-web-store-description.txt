WHAT BARRYGUARD DETECTS

- Serial ruggers: creators with a rug history are flagged automatically
- Insider networks: multiple top holders funded by the same wallet
- Bundle launches: creator bought before anyone else using Jito bundles
- Token-2022 Permanent Delegate: the extension that allows stealing tokens from any holder's wallet
- Post-rug tokens: large holder count but empty liquidity pool — already rugged
- Honeypots: tokens that can be bought but not sold


SCORING

Each check produces a score. The weighted average gives a total risk score from 1–99, plus three subscores. Result levels: Danger · High · Caution · Moderate · Low.
